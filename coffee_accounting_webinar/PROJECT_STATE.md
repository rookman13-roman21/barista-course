## Что подготовлено

- Страница собирается как набор HTML-блоков для Tilda в порядке `00 → 05`.
- `index.html` используется только для локального превью через VS Code Live Preview или локальный сервер.
- Страница создана в Tilda: pageid `148220666`, alias `/coffee_accounting_webinar`; блоки вставлены вручную.
- Блок `03` читает публичный JSON, обновляет дату/ссылку в видимых блоках и переключает CTA на `#waiting_list`, если событие прошло, не найдено или мест нет.
- Чтобы не было мигания старой даты, блок `00` скрывает дату и кнопки записи до применения JSON-синхронизации; после обновления данных блок `03` добавляет `body.mbs-caw-sync-ready`.
- Если публичный JSON недоступен, используется статический fallback: `4 июля 2026, 19:30` и исходная ссылка записи.
