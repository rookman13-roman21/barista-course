## Как проверить

1. Открыть `[slug]/index.html` через VS Code Live Preview.
2. Проверить desktop и mobile 390px.
3. Проверить кнопки `#consalt`, `#waiting_list`, `#order:...` без конфликтующих `id`.
4. Проверить запись/покупку по текущей механике страницы.
5. Проверить FAQ, smooth scroll, popup/modal, если есть.
6. Проверить SEO: canonical, meta, JSON-LD, один H1, `alt` у изображений.
7. После вставки в Tilda проверить опубликованную страницу вручную.
