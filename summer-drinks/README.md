# MBS Mixology Recipes prototype

Локальный статический прототип публичной витрины авторских рецептов напитков для сайта Московской школы бариста.

## Что это

- Первый уровень: каталог рецептов для будущей страницы `/summer_drinks`.
- Второй уровень: продающий preview рецепта `/summer_drinks/ice_tea_2026`.
- Позиционирование: рецепты участников и призёров MBS MIXOLOGY CUP, сезонные дропы и авторские подборки для кофеен.
- Интеграции с `Coffee_menu`, backend, оплатой, личным кабинетом и Tilda API пока нет.

## Файлы

| Файл | Назначение |
|---|---|
| `index.html` | Локальный preview каталога через iframe |
| `recipe-ice-tea-2026.html` | Локальный preview страницы рецепта через iframe |
| `tilda-blocks/catalog.html` | Самодостаточный HTML-блок каталога для Tilda |
| `tilda-blocks/recipe-ice-tea-2026.html` | Самодостаточный HTML-блок рецепта для Tilda |
| `PROJECT_STATE.md` | Паспорт решений, связей и ограничений |

## Публичные поля

Статический JS-объект в каталоге сделан под будущий safe-preview payload из `Coffee_menu`:

- `id`, `public_slug`, `title`, `category`, `group_name`;
- `volume_ml`, `price`, `public_description`, `image_url`;
- `author.name`, `author.avatar_url`, `author.bio`;
- `is_mixology`, `season`, `format`, `complexity`, `score`, `badges`;
- `taste`, `equipment_preview`, `included`.

## Что показываем до покупки

- фото, название, автор, категория, объём, цена;
- связь с MBS MIXOLOGY CUP и оценку, если доступна;
- сезон, формат кофейни, сложность, вкусовую идею;
- preview оборудования, чтобы кофейня понимала пригодность рецепта к своей станции;
- состав комплекта после покупки: техкарта, видео, ингредиенты, поставщики, личный кабинет, поддержка.

## Что не показываем до покупки

- граммовки и точный состав;
- полный процесс приготовления;
- точную техкарту;
- внутренние экономические расчёты;
- телефоны, email, Telegram, yClients ID, record ID;
- внутренние ID публикаций, review-поля, Битрикс-поля и служебные статусы.

## Проверка

Открыть локально:

```text
/Users/Romka/Downloads/All_Code/barista-course/summer-drinks/index.html
/Users/Romka/Downloads/All_Code/barista-course/summer-drinks/recipe-ice-tea-2026.html
```

Проверить:

- каталог показывает рецепты, авторов, цены, бейджи, форматы и CTA `Подробнее`;
- фильтры по автору, сезону и формату работают;
- страница рецепта не раскрывает граммовки и процесс;
- кнопка `Запросить доступ` показывает только локальное сообщение;
- сетевых запросов и корзины нет.

## Следующий этап

После утверждения визуального прототипа подключать `Coffee_menu` отдельной задачей:

- `GET /api/public/author-recipes` для каталога;
- `GET /api/public/author-recipes/{slug}` для страницы рецепта;
- покупка, оплата и выдача материалов через личный кабинет;
- связь с авторами, оценками и участиями MBS MIXOLOGY CUP.
