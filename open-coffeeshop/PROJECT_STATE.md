## 📝 Уроки из работы над страницей

| Проблема | Решение |
|---|---|
| Лейбл и h2 одинаковые | Лейбл = категория («Аудитория курса»), h2 = конкретный смысл («Для кого этот курс») |
| Две одинаковые кнопки в Hero | Оставлять одну CTA-кнопку `#CC2841` — если обе ведут на `#consalt` |
| aside короче левой колонки | Обёртка `align-items: stretch`, aside — `display: flex; flex-direction: column; flex: 1`, кнопка — `margin-top: auto` |
| Два блока с одинаковым фоном подряд | Менять фон секции или карточек одного из блоков |
| Тёмно-зелёный CTA `#417033` в финальном блоке | Заменить на `#E7F2E3` — финальный CTA всегда светло-зелёный |
| Ведущих трое, но курс индивидуальный | Оставлять только тех, кто реально ведёт этот курс |
| Schema.org не добавлен | Для лендинга курса обязателен `@type: Course` с `offers.price` |
