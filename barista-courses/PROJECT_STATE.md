# Barista Courses: состояние проекта

Этот файл оставлен как короткий указатель.

Актуальная база знаний по странице базового курса бариста находится здесь:

`PROJECT_KNOWLEDGE.md`

Именно `PROJECT_KNOWLEDGE.md` считать основным источником правды по:

- структуре Tilda-блоков;
- опубликованному состоянию;
- онлайн-записи;
- ценам и форматам курса;
- контактам;
- UI-решениям;
- проверкам безопасности;
- правилам, которые нельзя ломать при следующих правках.
