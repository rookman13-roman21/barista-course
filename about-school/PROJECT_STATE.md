## 📦 Tilda-блоки (готовы к вставке)

Каждый блок разбит в отдельный HTML-файл:
```
about-school/blocks/
  block-01-hero.html
  block-02-mission.html
  block-03-founder.html
  block-04-whofor.html
  block-05-trainers.html
  block-06-projects.html
  block-07-space.html
  block-08-reviews.html
  block-09-cta.html
```

Каждый файл содержит:
- HTML-комментарий с описанием блока вверху
- `<link>` Mulish и/или `<script>` Phosphor Icons (где нужно)
- `<section>` с `<style>` и разметкой — готово к вставке в Zero Block Tilda
